nmap scan

2017-03-28

掃同網段有那些 IP (用 ping 的)

nmap -sP 192.168.1.0/24
nmap -sP 192.168.1.1-254

OS

nmap -O 192.168.1.1

Port

nmap 192.168.1.1 -p21

全面掃描

nmap -A -T4 192.168.1.1

只列出網段清單

nmap -sL 192.168.0/24

TCP SYN

nmap -sS 192.168.1.1

TCP connect

nmap -sT 192.168.1.1

UDP

nmap -sU 192.168.1.1
